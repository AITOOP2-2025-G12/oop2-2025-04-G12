## 出力されるテキストファイルについて
- 文字起こしによって得られたテキストは、テキストファイルに出力されます
- ファイル名は、```audio.txt```です。
- 同じファイル名があった場合、```audio2.txt```, ```audio3.txt```...のように番号が増えていきます。

リポジトリの目的
音声を文字起こしして保存する

実行するのに必要なモジュール
main.py

Sagyo3.py
